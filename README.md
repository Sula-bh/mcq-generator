# 🧠 AI MCQ Generator with Evaluation

## 🌐 Live Demo

[![Open App](https://img.shields.io/badge/Streamlit-Live_App-ff4b4b?logo=streamlit)](https://mcq-generator-123.streamlit.app/)

An AI-powered application that generates Multiple Choice Questions (MCQs) from:

- 📄 Uploaded files (PDF/TXT)
- 📚 Given subject/topic

Built using **Streamlit + LangChain + Gemini (Google Generative AI)**.

---

## 🚀 Features

### 🎯 MCQ Generation

- Generate MCQs from **text files (PDF/TXT)**
- Generate MCQs from a **subject/topic**
- Customizable:
  - Number of questions
  - Difficulty level (Easy / Medium / Hard)

---

### 🤖 AI Evaluation

- Automatically reviews generated MCQs
- Fixes:
  - Incorrect questions
  - Wrong answers
  - Poor clarity
- Ensures high-quality output

---

### 📊 Quiz Interface

- Interactive quiz UI
- Select answers and submit
- Instant feedback:
  - ✅ Correct answers
  - ❌ Wrong answers
- Displays final score

---

### ⚙️ Robust System

- Structured JSON output parsing
- Error handling for invalid LLM responses
- Logging for debugging and traceability
- Caching for performance optimization

---

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **LLM**: Google Gemini (via LangChain)
- **Backend Logic**: Python
- **Parsing**: JSON Output Parser
- **Logging**: Python logging module
